## 🚀 使用方式 (用户)

1. 在 Muse AI 平台生成你创作的音乐, 并获取**分享链接** (形如 `https://h5.muse.top/song?id=xxxxxx`)。
2. 打开本工具 (本地 / GitHub Pages 链接均可)。
3. 首次使用需阅读并同意协议 (**滚动到底部并勾选**)。之后浏览器自动记住, 无需重复确认。
4. 粘贴分享链接, 按 Enter 或点 `[ DOWNLOAD ]`。
5. 浏览器内自动抓取元数据 → 音频 → 封面 → 打包 zip → 触发下载, 整个过程不经过任何后端。

## 🌐 一键部署到 GitHub Pages

整个仓库只需要 `index.html` 一个文件就能跑。

1. ⭐ 点个 Star (可选, 但作者会很开心)
2. 🍴 Fork 这个仓库到你的账号
3. ⚙️ 进 `Settings` → `Pages` → `Source` 选 `Deploy from a branch`
4. 🌿 Branch 选 `main` / `(root)`, 保存
5. ⏳ 等 1~2 分钟, GitHub 会给你一个 `https://<你的用户名>.github.io/<仓库名>/` 的链接
6. 🎉 打开链接, 粘贴 `https://h5.muse.top/song?id=xxxxxx` 即可

`app.py` / `requirements.txt` / `run.bat` 是 Python 版用的, **不影响静态部署**, 可以一起留着, 也可以删。

## 特性

- **终端复古风** — 磷光绿 `#00ff41` + CRT 扫描线 + 边缘暗角 + 等宽字体 + 闪烁光标
- **浏览器内打包** — JSZip 直接在页面里打成 zip (`mp3` + `cover.jpg` + `lrc` + `info.json` + `README.txt`), 不下到任何服务器
- **强约束免责弹窗** — 必须**滚动到底部**才启用勾选, 勾选后 `localStorage` **永久记录**, 不再重弹
- **Rickroll 彩蛋** — 拒绝协议有一定概率跳转到 B 站经典视频哦~ (透露: BV********VH)
- **会话内历史** — 元数据存 `localStorage`, Blob 留缓存 (一般刷新后失效, 等价于"3 天过期")
- **零依赖部署** — 单一 HTML 文件, 只外链一个 JSZip CDN
- **真实下载进度** — 用 `ReadableStream` 读 chunk, 字符进度条 `[####------] 40%` 实时更新
- **零后端** — 抓取、打包、下载全在用户浏览器里, 平台看到的是用户 IP 不是集中抓取服务器！

## 技术细节 (CORS 可行性)

静态版能跑通的关键是 **Muse 的 API + 音频 + 图片资源都允许浏览器跨域 fetch** (`Access-Control-Allow-Origin: *`):

| 资源类型 | 来源 | 跨域状态 |
|---|---|---|
| 元数据 API | `https://project-api.atmob.com/.../song/info` | 允许 |
| mp3 音频 | `*.cos.ap-*.myqcloud.com` (腾讯云 COS) | 允许 |
| 封面图片 | 平台 CDN | 允许 |

这意味着用户浏览器可以**直接 `fetch` 拿到 mp3 二进制**, 用 `JSZip` 在页面内打 zip, 然后 `URL.createObjectURL` + `<a download>` 触发下载, 全程不经过任何第三方代理。

## Python 版 (备选 · `app.py`)

> **此版本在服务端发起抓取**, 部署请注意: 你的 IP 会向 Muse 平台发起请求, 可能违反其 ToS。
> 仅推荐**个人内网自部署**做研究和备份用。

Python 版内嵌了同一份 HTML (逻辑与静态版完全一致), 优势是:

- **双层抓取** — `requests` 主路失败时自动回退到 Playwright 真实打开页面
- **磁盘持久化** — zip 落到 `downloads/`, 3 天后自动清理
- **任务调度** — 信号量限流 + 同 work_id 去重 + 进度加权

### 启动

```bash
# 一键启动 (Windows)
run.bat

# 或手动
pip install -r requirements.txt
python app.py
# 浏览器打开 http://127.0.0.1:48721
```

可选 Playwright 回退 (主路失败时才需要):

```bash
python -m playwright install chromium
```

> 注意: Python 版的免责弹窗用的是早期版本 (`sessionStorage` 3 分钟), 不带"滚动到底部"逻辑, 也不引用 `bc/1.txt`。如果你 fork 了这个仓库, 想统一体验, 可以用 `index.html` 替换 `app.py` 启动后的内嵌 HTML。
